# SparkFun ICM-20948 Arduino Library

This is based on the SparkFun library for the TDK InvenSense ICM-20948 Inertial Measurement Unit 9-Degree Of Freedom sensor as used on the [SparkFun 9DoF IMU Breakout - ICM-20948 (Qwiic)](https://www.sparkfun.com/products/15335).

This version uses the linux i2cdev system and removes all the arduino dependencies. It includes DMP support.
## Repository Contents

* [**/examples**](./examples) - Example sketches for the library (.ino). Run these from the Arduino IDE - these are here for reference only
* [**/src**](./src) - Source files for the library (.cpp, .h).
* [**DMP.md**](./DMP.md) - Information about the InvenSense Digital Motion Processor (DMP™)
